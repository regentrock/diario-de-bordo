# Diário de Bordo - PWA

Um aplicativo web progressivo (PWA) para registro de atividades diárias, que funciona offline e pode ser instalado na tela inicial do dispositivo.

## Funcionalidades

- Criar entradas com título, descrição e data
- Listar entradas registradas
- Remover entradas
- Interface responsiva para mobile e desktop
- Filtrar entradas por data
- Contador de entradas
- Funcionamento offline
- Instalação na tela inicial
- Persistência de dados no navegador

## Tecnologias Utilizadas

- HTML5
- CSS3 (Grid, Flexbox, Variáveis CSS)
- JavaScript (ES6+)
- Service Workers
- Web App Manifest
- LocalStorage API

## Como usar

### Online
1. Acesse o aplicativo através do navegador
2. Adicione entradas usando o formulário
3. As entradas serão salvas automaticamente
4. Instale o aplicativo clicando no botão "Instalar App"

### Offline
1. Após a primeira visita online, o aplicativo funcionará offline
2. Todas as funcionalidades estarão disponíveis
3. As entradas serão salvas localmente
4. Quando a conexão retornar, os dados permanecerão no dispositivo

## Instalação como PWA

### Chrome/Edge/Opera
1. Acesse o aplicativo
2. Clique no botão "Instalar App" no cabeçalho
3. Ou clique no ícone de instalação na barra de endereço

### Firefox
1. Acesse o aplicativo
2. Clique no menu (três pontos) e selecione "Instalar"

### Safari (iOS)
1. Acesse o aplicativo no Safari
2. Toque no ícone de compartilhar
3. Role para baixo e toque em "Adicionar à Tela de Início"
